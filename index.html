Route Concept Console

This route is a wearable piece of street typography. The run starts with a giant “U”, bends through neighborhood blocks to carve “MAD”, and closes with “BRO?” plus a dramatic question mark tail.

Style direction: inspired by the trollface meme spirit (playful, chaotic, self-aware), but designed with clean plotting logic so each character is readable on Strava map replay.

[U] ────── turn-by-turn arcs [MAD] ──── orthogonal block lettering [BRO?] ─ rounded corners + punctuation sweep ETA split: U 2.1 mi MAD 3.4 mi BRO? 4.2 mi
